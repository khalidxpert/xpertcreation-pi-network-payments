# XpertCreation Pi Network Payments for WooCommerce

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-GPL%20v2-green)
![WooCommerce](https://img.shields.io/badge/WooCommerce-5.0%2B-purple)

Accept Pi Network cryptocurrency payments in your WooCommerce store.

## Features

- ✅ Accept Pi Network payments at checkout
- ✅ Easy setup - just add your Pi App ID and API Key
- ✅ Sandbox/Testnet mode for testing
- ✅ Automatic payment verification
- ✅ Order status auto-update after payment
- ✅ Live Pi exchange rate support
- ✅ Mobile friendly - works in Pi Browser

## Installation

1. Download the plugin
2. Upload to `/wp-content/plugins/pi-pay-woocommerce/`
3. Activate from WordPress Plugins menu
4. Go to **WooCommerce > Settings > Payments > Pi Network Payment**
5. Enter your Pi App ID and API Key
6. Enable Sandbox mode for testing

## Requirements

- WordPress 5.0+
- WooCommerce 5.0+
- PHP 7.4+
- Pi Network Developer Account

## Getting Pi Credentials

1. Visit [Pi Developer Portal](https://developers.minepi.com)
2. Create a new app
3. Copy your **App ID** and **API Key**
4. Paste in plugin settings
